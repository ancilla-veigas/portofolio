This is my personal portfolio website built with **HTML, CSS, and JavaScript**.  
It showcases my skills, projects, certifications, and contact information in a modern, responsive design.

## Features
- Glassmorphism UI with smooth animations
- Dark/Light mode toggle
- Typing animation in hero section
- Project filtering and QR code generator
- Sections: Home, About, Skills, Certifications, Projects, Contact

## Technologies
- HTML5, CSS3 (Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- QRCode.js library

## Deployment
The site is hosted online using GitHub Pages / Netlify / Vercel.  
You can view it live here: ephemeral-pavlova-312420.netlify.app
